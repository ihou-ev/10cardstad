# 10カードスタッド - ゲームルール

## 1. ゲーム概要

- 52枚（ジョーカーなし）のトランプを使用
- プレイヤーは**5人固定**
- 各プレイヤーに**10枚**配る（合計50枚使用、残り2枚は不使用の死に札）
- 最終的に各プレイヤーは、自分の10枚から任意の**5枚**を選び、通常のポーカー役で勝負
- 最も強い役のプレイヤーが勝利
- 同着（完全同強度）の場合はポットを等分

---

## 2. 用語

| 用語 | 説明 |
|------|------|
| **ドアカード（Door Cards）** | 最初から表向きで配られる5枚 |
| **ホールカード（Hole Cards）** | 最初は裏向きの5枚（後で公開される） |
| **公開カード（Revealed Set）** | そのプレイヤーについて現在表向きになっているカード集合（初期はドアカード5枚、以降増える） |
| **現在強さ（Current Strength）** | 公開カードだけで作れる"現時点の最強5枚役"の強さ |
| **最終強さ（Final Strength）** | 10枚すべて公開後、10枚から選べる最強5枚役の強さ |

---

## 3. 配り方（初期状態）

1. デッキをシャッフル
2. 各プレイヤーに10枚配る：
   - 最初の**5枚は表向き**（ドアカード）
   - 残り**5枚は裏向き**（ホールカード）
3. 配られなかった2枚はゲーム中使用しない（誰も見ない）

> **注意**: ホールカード5枚は「誰にも見えない」。プレイヤー本人も公開されるまで内容を知らない（めくるまで不明）。

---

## 4. ゲーム進行（"最弱から公開"フェーズ）

### 4.1 初期評価

- 各プレイヤーは公開カードが5枚あるため、その5枚で役を判定できる
- この時点の役強度を **Current Strength** とする

### 4.2 公開ループ（最弱から1枚公開）

以下を、ホールカードが尽きるまで繰り返す（**合計5ラウンド**）：

1. 全プレイヤーの **Current Strength** を計算する
   - 公開カードが5枚の人：その5枚が評価対象
   - 公開カードが6枚以上の人：その公開カード集合から**最強の5枚**を選び評価
2. Current Strength が**最も弱いプレイヤー**を選ぶ（= "最弱"）
3. 最弱プレイヤーは、残っているホールカードから**1枚を表向き**にする
4. 公開カードが増えたことで Current Strength が更新され、次の最弱判定に影響する
5. 全員のホールカードが0枚になる（= 全員10枚公開）まで続行

### 4.3 最弱が複数いる場合（同点最弱）

- 同点で最弱が複数いる場合は、**その全員が同時に1枚ずつ公開**する

---

## 5. 最終勝負

1. 全員の10枚が公開されたら、各プレイヤーは自分の10枚から**最強の5枚**を選ぶ
2. **Final Strength** を計算
3. Final Strength の最大が勝者
4. 同着（完全一致）の場合はポット等分

---

## 6. 役の定義（通常の5枚ポーカー）

### 役の強さ順（強い順）

| 順位 | 役名 | 説明 |
|:----:|------|------|
| 1 | **ストレートフラッシュ** | 同じスートの連続5枚 |
| 2 | **フォーカード** | 同じランク4枚 + 1枚 |
| 3 | **フルハウス** | スリーカード + ワンペア |
| 4 | **フラッシュ** | 同じスート5枚（連続でない） |
| 5 | **ストレート** | 連続5枚（スート不問） |
| 6 | **スリーカード** | 同じランク3枚 + 2枚 |
| 7 | **ツーペア** | ペア2組 + 1枚 |
| 8 | **ワンペア** | 同じランク2枚 + 3枚 |
| 9 | **ハイカード** | 上記いずれにも該当しない |

### ストレートのAの扱い

- Aは通常**ハイ**（…Q-K-A）
- **A-2-3-4-5** はストレートとして認め、**最弱ストレート**扱い

### スート順位

- スートに強弱は**付けない**（フラッシュ同士はランク比較のみ）
